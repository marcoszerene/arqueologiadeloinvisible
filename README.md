Analizador de Arqueología Visual - FAD UNCuyo

Esta es una herramienta interactiva diseñada para la cátedra de Diseño Gráfico F (López Dupertuis) de la Facultad de Artes y Diseño de la Universidad Nacional de Cuyo.

El objetivo de la aplicación es facilitar el diagnóstico de patologías visuales en el entorno urbano mediante el mapeo de registros de campo y la generación de gráficos de radar (araña) para visualizar puntos de dolor críticos.

🚀 Funcionalidades

Mapeo de Capas: Permite agregar hasta 5 registros comparativos (Registros A, B, C, etc.).

Variables de Análisis: Evalúa Sintonía, Fricción, Obsolescencia, Coherencia e Inclusividad.

Gráfico Dinámico: Visualización en tiempo real de las patologías detectadas.

Generación de Informe: Formato optimizado para impresión en A4 o guardado como PDF con membrete institucional.

Análisis de Hallazgos: Espacios dedicados para la redacción del Insight y la pregunta How Might We (HMW).

🛠️ Cómo usarlo

Ingresa los puntajes (1 al 3) para cada una de las 5 variables en las diferentes capas de registro.

Identifica el "Hallazgo Principal" que la herramienta calcula automáticamente basándose en los promedios.

Redacta las conclusiones en los campos de texto inferiores.

Haz clic en "IMPRIMIR REPORTE A4" para generar el documento final de entrega.

📦 Instalación

No requiere instalación. Es un proyecto Single File que puedes ejecutar abriendo el archivo index.html en cualquier navegador moderno.
